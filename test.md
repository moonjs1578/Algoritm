백준 17626번 "Four Squares" 문제네요! 라그랑주의 네 제곱수 정리를 활용하는 문제입니다.

핵심 아이디어:

모든 자연수는 최대 4개의 제곱수의 합으로 표현 가능 (라그랑주의 정리)
우리는 최소 개수를 찾아야 하므로 1개, 2개, 3개로 가능한지 차례로 확인
접근 방법:

1개로 가능한지 체크: n이 완전제곱수인지 확인
√n이 정수면 답은 1
2개로 가능한지 체크: i² + j² = n 형태로 표현 가능한지
i를 1부터 √n까지 순회하면서 (n - i²)이 완전제곱수인지 확인
3개로 가능한지 체크:
비슷하게 i² + (n - i²을 2개 제곱수로) 형태 확인
또는 4개가 필요한 경우를 먼저 제외
4개: 위에서 걸러지지 않으면 4개
최적화 팁:

라그랑주 정리에 의해 4^a(8b+7) 형태의 수만 정확히 4개가 필요
이를 미리 체크하면 3개 확인을 스킵할 수 있음
시간복잡度:

O(√n)로 충분히 해결 가능 (n ≤ 50,000)
DP로도 풀 수 있지만, 위 방법이 더 효율적입니다. 구현해보세요!

이해하기 쉽고 안정적입니다
방법 2: 수학적 접근

1개→2개→4개→3개 순서로 체크
더 빠르지만 이론을 알아야 합니다
초보자라면 **방법 1(DP)**을 추천드려요. 코드도 더 짧고 이해하기 쉽습니다!

어떤 수 i를 만들려면, i보다 작은 제곱수 j²를 하나 선택하고
나머지 (i - j²)를 만드는 최소 개수에 1을 더하면 됩니다
이 중 최솟값을 선택!
